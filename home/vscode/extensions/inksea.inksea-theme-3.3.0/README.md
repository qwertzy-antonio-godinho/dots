## My Settings

Font and indent rainbow colors.

{
    "editor.fontFamily": "Input Mono",
    "indentRainbow.colors": [
        "rgba(255, 255, 255, 0)",
        "rgba(255, 255, 255, 0.03)",
        "rgba(255, 255, 255, 0.06)",
        "rgba(255, 255, 255, 0.1)",

        "rgba(255, 255, 255, 0.03)"
    ]
}
